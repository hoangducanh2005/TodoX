mongodb+srv://... của MongoDB Atlas cần giải tên host qua DNS SRV record.

Khi bạn thêm (và không comment) hai dòng:

thì Node sẽ dùng Google DNS để giải tên, và cluster0.7mdbnbk.mongodb.net được tìm thấy, nên kết nối thành công.

Nếu không có dòng đó, Node sẽ dùng DNS mặc định của hệ thống bạn. Nếu DNS mặc định không trả về được record SRV cho Atlas, thì sẽ bị:
Error connecting to MongoDB

Tóm lại
Không phải vì hai dòng đó “kết nối được database”, mà vì chúng sửa lại DNS resolver trước khi Mongoose kết nối.

 Toast
 Trong thư viện Sonner, Toast chính là cái "thông báo nhỏ" (pop-up) nhảy lên ở góc màn hình để phản hồi cho người dùng biết 
 một hành động nào đó đã thành công, thất bại hoặc đang xử lý.
 2. Các thành phần chính khi dùng Sonner
Để dùng được Toast, Sonner chia làm 2 phần:

<Toaster />: Là "cái khay" chứa các thông báo. Bạn chỉ cần đặt nó một lần duy nhất ở file cha (như App.jsx hoặc Layout.jsx).

toast(): Là hàm để bạn "bắn" ra thông báo bất cứ khi nào cần.

import { Toaster, toast } from 'sonner';

function App() {
  return (
    <div>
      <Toaster position="top-right" richColors /> {/* Đặt cái khay ở đây */}
      
      <button onClick={() => toast.success('Đã gửi tin nhắn!')}>
        Gửi ngay
      </button>
    </div>
  );
}

Tim hieu them ve thu vien cva, cac variant
cva là thư viện class-variance-authority, thường dùng trong shadcn/ui để quản lý class Tailwind theo nhiều kiểu khác nhau.

Hiểu đơn giản:

cva giúp một component có nhiều “biến thể giao diện” như primary, secondary, destructive, small, large...

Ví dụ Button có thể có nhiều kiểu:

<Button variant="default">Save</Button>
<Button variant="destructive">Delete</Button>
<Button size="sm">Small</Button>

Vì sao cần cva?

Không dùng cva, bạn phải viết kiểu này rất rối:

<button className={variant === "destructive" ? "bg-red-500" : "bg-blue-500"}>

Dùng cva thì gọn hơn:

<Button variant="destructive" size="lg">
  Delete
</Button>

Tóm gọn

cva dùng để tạo nhiều phiên bản giao diện cho cùng một component.

Ví dụ với Button:

variant = màu/kiểu button
size = kích thước button

Nó rất hay dùng trong shadcn để component vừa đẹp, vừa dễ tái sử dụng.
